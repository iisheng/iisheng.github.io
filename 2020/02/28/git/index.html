Git简单介绍

Git是一个分布式版本控制软件，最初由Linus Torvalds创作，于2005年以GPL发布。最初目的是为更好地管理Linux内核开发而设计。

Git文件状态变化图解

Git各个区域流程图解

  • Workspace：工作区
  • Staging：暂存区
  • Local Repository：本地仓库（可修改）
  • Local Remote Repository：本地远程仓库（本地的远程仓库的镜像，不可修改）
  • Remote：远程仓库
