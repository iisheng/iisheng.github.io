Linux命令速查笔记

发表于 2019-02-25 更新于 2019-10-20 分类于 😳工具

cd 进入指定目录

ls 列出当前目录文件

个人感觉 ll -h 比较好用

mv 修改文件名字或移动位置

rm 删除

cp 复制

kill 杀进程

chown 修改文件所属用户或用户组

chmod 给文件设置别人访问权限

cat 查看文件或写到指定文件内容

scp 终端远程下载上传文件

netstat 打印网络信息状态

top 实时系统的查看系统运行情况

htop 和top类似功能

sed 处理文件并在控制台打印

这个东西很多功能的，暂时不详细学习

less 从头查看日志

more 从头查看日志

tail 动态查看文件尾部内容

head 显示文件开头内容

rar 解压rar文件或压缩为rar文件

unzip 解压zip文件或压缩为zip文件

service 启动查看进程状态

systemctl 启动查看进程状态

安装、卸载软件

dpkg

shutdown

df

du

find 查找

grep 全面搜索正则表达式并把行打印出来

一般配合别的命令一起使用效果很强大

ps 显示当前进程状态

lsof 查看进程打开的端口，查看打开文件的进程

iconv 转换文件编码

curl 文本传输工具
